# SEO Content Engine

## Goal

Build content only where there is measurable demand or commercial value.

Do not fill the blog with generic posts. Use Search Console, service priority,
city priority, and conversion data.

## Inputs

- Search Console query/page data
- existing sitemap and blog index
- service list
- city/area list
- CRM lead quality by service/city
- Google Ads search terms when available

## Opportunity Types

### Type 1: Position 4-20 with impressions

Queries already have visibility but need content/CRO work.

Action:

- improve existing page
- add FAQ
- add internal links
- improve title/meta
- add city/service block

### Type 2: High impressions, low CTR

Page is visible but snippet is weak or intent mismatch exists.

Action:

- rewrite title/meta
- add stronger offer
- use local proof
- align H1 with query intent

### Type 3: Missing page

Search demand exists but no dedicated page exists.

Action:

- create service-city page
- create guide
- create calculator/tool page

### Type 4: Ads term with lead quality

Paid search term creates good leads. Turn it into SEO content.

Action:

- create SEO page
- add internal links from service page
- use Ads copy as CTA hypothesis
